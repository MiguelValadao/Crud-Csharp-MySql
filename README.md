 ### CRUD em C#, MySQL & .NET Framework

 ---
 
Neste projeto, estudei como conectar e executar ações em um banco de dados com uma interface Forms do .NET em C#, aprofundei-me ainda mais em programação orientada a objetos e otimizei a aplicação, mantendo como prioridade a segurança (evitando SQL injections) e a usabilidade (mantendo a UI o mais intuitiva possível).
Estou trabalhando em mais alguns projetos com um nível de complexidade um pouco maior.
